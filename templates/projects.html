Calico

A school-wide online chat app for students to anonymously communicate and vent with specially trained students built with Python Django and Bootstrap.

  • ✅ Coordinated with school counselors and Psychology club to develop and operate the website.
  • ✅ Constructed procedures in different situations based on meetings with school counselors and hotline responders
  • ✅ Trained a dozen students to be listeners.
  • ✅ Promoted with QR codes that connects to the website in student bathroom stalls.
  • ✅ Organized and surveyed students for their mental health-related problems and provided feedback.
Technologies used
Python HTML CSS Js

SharkVolunteers

A WeChat Mini-Program built with Django for teachers to request student volunteer work. Majorly resolved the problem of students being unable to meet their 100-hour community service hour requirements and teachers not having resources to request help when needed.

  • ✅ Responsive interface to see what volunteer work is available (tutor opportunities, straightening classrooms).
  • ✅ Easy to use and intuitive UI for teachers to request help.
  • ✅ Automatically logged community service hours linked to Academic Affairs teachers.
Technologies used
HTML Js Python

BookTank

A WeChat Mini-Program built with Django for students to buy/trade secondhand textbooks. The program aimed to eliminate the anxiety that students and parents had every year around the need to buy and sell their textbooks.

  • ✅ Up-to-date yearly book lists.
  • ✅ Real-time online chat for students and parents to coordinate handoffs.
  • ✅ Responsive picture upload and listing interfaces.

Achieved over 100 users per day in summer and winter vacations.

Technologies used
Python HTML CSS Js
